# AI Substitute Teacher System

A full-stack sample project implementing an "AI Substitute Teacher" system.

Stacks:

- Frontend: React (Vite) + TypeScript
- Backend: Node.js + Express + TypeScript
- AI Services: Python + FastAPI
- Database: MongoDB (Mongoose)
- Auth: JWT (Role-based)

Features:

- Student: anonymous one-click "Ask Doubt" submissions
- Teacher: session control, view hard questions, answer/dismiss
- AI: Question Analyzer (category + confidence), Answer Engine (teaching-style answers)
- FAQ knowledge base (MongoDB)

See `./README_DEV.md` for setup and run instructions.
